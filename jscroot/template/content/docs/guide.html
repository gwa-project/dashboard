User Guide

How to use the GWA Project Dashboard

Getting Started

Welcome to the GWA Project Dashboard. Here's how to get started:

  1. Login - Use your Google account to sign in
  2. Dashboard - View your project overview and statistics
  3. Profile - Manage your account settings
  4. Settings - Configure system preferences

Navigation

Use the sidebar menu to navigate between different sections:

  • 🏠 Home - Dashboard overview
  • 👤 Profile - Account management
  • ⚙️ Settings - System configuration
  • 📚 Docs - Documentation and help

Support

If you need help, please contact support or check the API documentation.
