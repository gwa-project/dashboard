API Documentation

GWA Project API Reference

Authentication

All API requests require authentication using PASETO tokens.

Endpoints

  • POST /auth/login - User authentication
  • GET /user/data - Get user information
  • GET /project/data - Get project data
  • POST /project/create - Create new project

Response Format

{
  "status": 200,
  "message": "Success",
  "data": {}
}
