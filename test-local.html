How to Access Login

To access the full application:

  1. Option 1: Run a local HTTP server: 
    cd dashboard
python -m http.server 8080
# Then open: http://localhost:8080
  2. Option 2: Deploy to web server
  3. Option 3: Use Live Server extension in VS Code
Note: ES6 modules require HTTP server due to CORS policy. Direct file:// access won't work with the full jscroot system.
